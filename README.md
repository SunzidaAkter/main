#Employee Management System

The Employee Management System is a MySQL database project used to manage employee information in an organization. This project helps store and organize employee data in a simple and structured way. It includes different sections to manage departments, employee details, attendance, leave requests, salary information, training, and performance. The project also shows how database tables are connected using SQL.

### Features 

* **Department & Role Management**
  Stores department names and job roles of employees.

* **Employee Management**
  Stores basic employee details like ID, name, contact, salary, and joining info.

* **Attendance Management**
  Records whether employees are present, absent, or late.

* **Leave Request Management**
  Manages employee leave applications and their status.

* **Payroll Management**
  Stores salary details, bonuses, deductions, and payment status.

* **Training Management**
  Keeps record of employee training courses.

* **Performance Evaluation**
  Stores employee performance ratings.

* **Database Connection**
  Connects all tables using keys to keep data related and organized.
